About Me

Hello everyone! My name is Eric Tong, and I'm a junior majoring in Computer Science at Carnegie Mellon. I'm currently looking for an Summer 2022 software engineering internship.

I've worked with Python, Java, C/C++, and Go. I have experience in research, personal projects, and software engineering internships. This summer, I've been working on a research project in complexity theory at CMU funded by the Summer Undergraduate Research Fellowship program.

Besides Computer Science, I'm interested in teaching. I've been a Teaching Assistant for CMU's 15-210 Parallel and Sequential Data Structures and Algorithms class for two semesters. Drawing from my own experience competing in the USA computing Olympiad, I've also mentored students both one-and-one and in a classroom environment, teaching them algorithmic concepts and problem-solving techniques.

Projects

Here are some of the projects I've worked on (some can be viewed on Github).

CMU Schedule Optimizer
A tool to help students optimize their class schedules based on course registration data and a variety of customizable user metrics. Backend written in Python and integrated with HTML/CSS frontend with Flask. Awarded Sandia National Lab's Sponsor's Choice Award.

CMU Lost and Found Website
Website to consolidate lost and found resources onto a single easily accessible site. Includes various admin features and filtering ccapabilities. Built using React and worked on in subteam of the ScottyLabs Tech Committee.

Wikipedia Question and Answering System
A question answering system written in Python that can answer natural language factoid questions about a Wikipedia article, as well as generate coherent questions. Uses natural language processing libraries, including NLTK, CoreNLP, and spaCy.

CMUPrinter
Android App that uses GPS and a printer database to help students locate the closest printer on campus that meets their printing requirements. Developed using Android Studio and Java.
